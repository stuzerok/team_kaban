## Баги
Баг №1: Доступно для взаимодействия 16 регистров но в докуменатции заявлено о 10 регистрах
Для воспроизведения  запустить test_stress_1000

Баг №2: Значение LCR(3) по сбросу 0x00 вместо ожидаемого 0x01 
Для воспроизведения  запустить bug0_test

Баг №3: Регистр IER(4) доступен для записи [31 : 4] хотя согласно спецификации доступны только [3:0]
Для воспроизведения  запустить bug0_1_test

Баг №4: Регистр IIR(5) доступен для записи но согласно спецификации он read only
Для воспроизведения  запустить bug0_2_test

Баг №5: Регистр LSR(7) доступен для записи но согласно спецификации он read only
Для воспроизведения  запустить bug0_2_test

Баг №6: Регистр MSR(9) доступен для записи но согласно спецификации он read only
Для воспроизведения  запустить bug0_2_test

Баг №7: Значение IIR(5) по сбросу 0x00 вместо ожидаемого 0x8C
Для воспроизведения  запустить bug0_test

Баг №8: Значение LSR(7) по сбросу 0x00 вместо ожидаемого 0x60
Для воспроизведения  запустить bug0_test

Баг №9: Запись в регистр LCR (3) блокирует чтение IER (4)
Для воспроизведения  запустить bug1_test

Баг №10: Невозможность отправки данных с значением data >= 0x00010000. Данные записываются не корректно
Для воспроизведения  запустить bug2_test

Баг №11: Запись в регистр DLBR_DLL (2) любого значения с data[7 : 0] = 0x42 приводит к потере данных data [31 : 6]
Для воспроизведения  запустить bug3_test

Баг №12: Биты регистра RBR_THR (0) [31 : 6] должны быть зарезервированы. Но они доступны для записи
Для воспроизведения  запустить bug4_test

Баг №13: Биты регистра FCR(6) [5:3] доступны для записи хотя являются зарезервироваными 
Для воспроизведения  запустить bug5_test

Баг №14: Биты регистра MCR(8) [7:4] доступны для записи хотя являются зарезервироваными 
Для воспроизведения  запустить bug6_test

Баг №15: Бит LCR[7] = 0 (3) не блокирует запись в DLL/DLM
Для воспроизведения  запустить bug0_3_test

## Запуск (пошагово)

```bash

# 1. Установи зависимости
py -m venv .venv
.venv\Scripts\activate.bat
python -m pip install streamlit pytest-cov pylint pytest
python generate_results.py
# 2. Распакуй черный ящик (НЕ открывать riscv_reg_block.py!)
python -m pytest tests/ --cov=riscv_reg_block --cov-report=html:coverage_report/
python3 loader.py

# 3. Запусти тесты + coverage
python -m pytest tests/ --cov=riscv_reg_block --cov-report=html:coverage_report/

# 4. Проверка кодстайла YADRO
pylint -m *.py tests/ dashboard.py  # Цель: ≥8.0/10

# 5. Dashboard с live метриками
python -m streamlit run dashboard.py
